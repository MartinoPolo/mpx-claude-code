# CSS & Styling

- 60-30-10 rule: 60% neutral backgrounds, 30% secondary UI, 10% accent
- Backgrounds: white, off-white, light cool tones only. Never red/orange/yellow/purple/pink
- Text: dark on light (#1f2937, #111827), light on dark (#f9fafb). Minimum 4.5:1 contrast (WCAG AA)
- Hot colors (red, orange, yellow): errors, warnings, destructive actions only. Never decorative
- Gradients: minimal color shift, same family, linear preferred. Never mix hot and cool
