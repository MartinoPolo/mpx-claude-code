MCP dependencies — remote/scheduled viability:
● chrome-devtools-tester — Chrome DevTools MCP: local-only, needs running Chrome GUI. Not viable in cloud tasks.
● context7-docs-fetcher — Context7 MCP (HTTP/SSE): works in cloud via connector
● issue-analyzer, issue-finder — gh CLI only (works everywhere)

Alternative for cloud tasks: Playwright MCP runs headless, works in VMs, cross-browser. Viable replacement for Chrome DevTools in scheduled execution.
