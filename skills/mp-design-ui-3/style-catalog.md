# Style Catalog

Curated design styles for variant generation. Each style defines typography, colors, layout philosophy, density, and motion approach. Based on [TypeUI](https://github.com/bergside/typeui).

When auto-selecting styles, maximize distance across these axes:
- **Theme** (light vs dark)
- **Typography** (serif vs sans vs mono vs display)
- **Density** (airy vs medium vs dense)
- **Mood** (professional vs playful vs raw vs elegant)
